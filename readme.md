# get-ip [![Build Status](https://travis-ci.org/alonalon/get-ip.svg?branch=master)](https://travis-ci.org/alonalon/get-ip)

> My posh module


## Install

```
$ npm install --save get-ip
```


## Usage

```js
const getIp = require('get-ip');

console.log(getIp());
//=>['192.168.9.104', '192.168.10.35']
```
